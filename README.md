# Typescript+Node.js+jest

## Get start

```:
npm install
```

## 動かし方

```:
npm start
```

## テストの動かし方

```:
npm test
```

- テストは```./__tests__```配下に入れること
- テストのファイルは```**.test.ts```という命名にすること
- Pathのエイリアスの設定をしているので、テスト対象をimportするときは```./src```からのパスを記述で良い

## コンパイル対象

```./src```配下

## アウトプット先

```./lib```
